Black Eyed Lucy Complex

Allelic locus: Fire, Sulfur, Disco, Flame, Vanilla.

Any two copies (homo- or hetero-allelic) = 🖤 Black Eyed Lucy.
Single copy = single-gene visual (e.g. Fire).

Spider Complex

Allelic locus: Spider, Spotnose, Chocolate, Cypress, Wookie, Blackhead, Champagne, Hidden Gene Woma, Saar, Sable, Shadow.

Single copy = visual. Same-gene homo = Super.
Spider/Spider & Champagne/Champagne = lethal.
Cross-gene compounds show both names.

BEL Complex

Allelic locus: Mojave, Lesser/Butter, Phantom, Mystic, Russo, Honey, Special, Bamboo.

Any two BEL copies (homo- or hetero-allelic) = ✨ Blue-Eyed Leucistic.
Single copy = single-gene visual (e.g. Mojave).

Yellow Belly Complex

Allelic locus: Yellow Belly, Asphalt, Gravel, Specter, Spark.

YB/YB = ✨ Ivory.
Same-gene homozygous = Super form.
Cross-gene compound = both visuals expressed.

8 Ball Complex

Allelic locus: Cinnamon, Black Pastel, Enchi, Red Axanthic (HRA).

All are codominant — 1 copy = visual, homo = Super.
Cin/BP = Cinnamon Black Pastel.
Cross-gene compounds show both names.

Albino Complex

Allelic recessive locus.

Albino/Albino = Visual Albino.
Candy/Candy = Visual Candy.
Albino/Candy = ✨ Candino (compound visual).

Crypton (Clown / Cryptic)

Allelic recessive locus.

Clown/Clown = Visual Clown.
Cryptic/Cryptic = Visual Cryptic.
Clown/Cryptic = ✨ Crypton (compound visual).

T+ Albino (Caramel / Ultramel)

Allelic recessive locus.

Caramel/Caramel = Visual Caramel Albino.
Ultramel/Ultramel = Visual Ultramel.
Caramel/Ultramel = T+ Albino compound.
